# 📸 Implement Photo Upload untuk Pemilik Pesantren

## 🎯 Summary
Menambahkan fitur upload foto utama dan galeri foto untuk pemilik pesantren saat membuat atau mengedit data pesantren di dashboard.

## ✨ Changes Made

### Frontend (`PemilikDashboard.vue`)

#### 1. **Form Upload UI**
- Added foto utama upload input with preview
- Added galeri foto upload (multiple files, max 5)
- Real-time preview sebelum submit
- Remove individual gallery photos
- File size validation (max 5MB)

#### 2. **State Management**
```javascript
const fotoFile = ref(null)          // Main photo file
const fotoPreview = ref('')         // Main photo preview
const galleryFiles = ref([])        // Gallery files array
const galleryPreview = ref([])      // Gallery previews array
```

#### 3. **Upload Handlers**
- `onFileChange()` - Handle main photo upload
- `onGalleryChange()` - Handle multiple gallery uploads
- `remove_gallery()` - Remove individual gallery photo
- Validation for file size (5MB) and count (max 5)

#### 4. **FormData Submission**
- Changed from JSON to FormData for multipart upload
- Appends `foto_utama` file
- Appends gallery files as `galeri_0`, `galeri_1`, etc.
- All other fields remain the same

#### 5. **Edit Mode Enhancement**
- Loads existing foto_utama preview
- Loads existing foto_galeri previews
- Properly handles edit vs create scenarios

### Navbar Enhancement (`AppNavbar.vue`)
- Fixed icon color to white (`text-white`)
- Maintained blue background (`bg-primary`)

## 📝 Technical Notes

### File Upload Flow
```
User Selects File
  ↓
Preview (URL.createObjectURL)
  ↓
Submit (FormData)
  ↓
Backend (@fastify/multipart)
  ↓
Save to uploads/pesantrenImages/
  ↓
Store filename in DB
  ↓
Display in Detail Page
```

### FormData Structure
```javascript
{
  nama: 'Pesantren XYZ',
  province: 'Jawa Barat',
  kota: 'Bandung',
  foto_utama: File,           // Main photo
  galeri_0: File,             // Gallery photo 1
  galeri_1: File,             // Gallery photo 2
  // ... other fields
}
```
