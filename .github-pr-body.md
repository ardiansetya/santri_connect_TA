## Summary

Implementasikan semua halaman authenticated frontend dengan **role-based dashboard** dan **form pendaftaran**.

## 🎨 Komponen Baru

### Dashboard (Role-Based)
| Role | Komponen | Fitur Utama |
|------|----------|-------------|
| **Pendaftar** | `PendaftarDashboard.vue` | Stats, riwayat pendaftaran, quick actions |
| **Pemilik** | `PemilikDashboard.vue` | CRUD pesantren sendiri, modal form |
| **Superadmin** | `AdminDashboard.vue` | Stats, manajemen pendaftaran, export Excel |

### Pendaftaran Form
- **Route**: `/pesantren/:id/daftar`
- 3 section: Data Pribadi, Data Orang Tua, Dokumen
- File upload: Foto KTP, Pas Foto, Kartu Keluarga
- Success state dengan nomor pendaftaran
