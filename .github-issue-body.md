# Fase 1: Implementasi Halaman Publik Frontend

## 📋 Deskripsi
Mengimplementasikan semua halaman publik frontend yang sebelumnya masih berupa placeholder/stub. Menggunakan **Bootstrap 5** untuk styling sesuai dengan library yang sudah terinstall.

## 🎯 Scope

### Halaman yang Diimplementasi

| Halaman | Route | Status |
|---------|-------|--------|
| **Pesantren List** | `/pesantren` | ✅ Implementasi |
| **Pesantren Detail** | `/pesantren/:id` | ✅ Implementasi |
| **Rekomendasi** | `/rekomendasi` | ✅ Implementasi |
| **Compare** | `/compare` | ✅ Implementasi + Fix Bug |
| **Track Status** | `/track` | ✅ Implementasi |

### Fitur per Halaman

#### 1. Pesantren List (`/pesantren`)
- Grid card layout responsive (Bootstrap `row-cols-*`)
- Filter: Provinsi, Kurikulum
- Sorting: Nama, Tahun Berdiri, Jumlah Santri, Biaya
- Pagination
- Tombol "Bandingkan" terintegrasi dengan Compare Store
- Hover effect pada card

#### 2. Pesantren Detail (`/pesantren/:id`)
- Breadcrumb navigation
- Layout 2 kolom: detail utama + sidebar
- Section: Informasi Umum, Biaya, Fasilitas, Kontak
- Info pendaftaran dari API
- Tombol tambah ke compare list
- Sticky sidebar

#### 3. Rekomendasi (`/rekomendasi`)
- Form input: Budget, Provinsi, Fasilitas (checkbox)
- Hasil dengan score breakdown (Budget 40%, Lokasi 30%, Fasilitas 30%)
- Progress bar untuk visualisasi score
- Info card cara kerja scoring
- Badge untuk kurikulum, biaya, jumlah santri

#### 4. Compare (`/compare`)
- Tabel perbandingan side-by-side (Bootstrap `table`)
- Load data dari Compare Store (Pinia)
- Hapus individual / hapus semua
- Badge kurikulum berwarna
- Auto-refresh saat store berubah

#### 5. Track Status (`/track`)
- Form input nomor pendaftaran
- Timeline visual proses (Pending → Diproses → Selesai)
- Detail pendaftaran dalam card grid
- Badge status berwarna (warning/info/success/danger)
- Catatan admin (jika ada)

### Tambahan Infrastruktur

| File | Perubahan |
|------|-----------|
| `src/services/index.js` | Tambah `pendaftaran.create()`, `admin.*`, `pemilik.*`, `wilayah.*` endpoints |
| `src/stores/compare.js` | **Baru** - Pinia store untuk manage daftar bandingkan (persist localStorage) |
| `src/main.js` | Import Bootstrap CSS |
| `backend/swagger.json` | **Baru** - OpenAPI 3.0.3 spec lengkap |
| `backend/src/server.js` | Register `@fastify/swagger` + `@fastify/swagger-ui` |

## 🛠 Tech Stack
- **Vue 3** - Composition API (`<script setup>`)
- **Bootstrap 5** - Grid, Cards, Forms, Tables, Badges, Progress, Alerts
- **Pinia** - State management (auth store + compare store)
- **Vue Router** - Navigation
- **Axios** - HTTP client dengan interceptor
