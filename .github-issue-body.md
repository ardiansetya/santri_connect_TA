# 📸 Feature: Photo Upload untuk Pemilik Pesantren

## 🎯 Deskripsi
Menambahkan kemampuan bagi pemilik pesantren untuk mengupload foto utama dan galeri foto saat membuat atau mengedit data pesantren di dashboard.

## ✨ Fitur yang Ditambahkan

### 1. **Upload Foto Utama**
- Input file untuk foto utama pesantren
- Preview foto sebelum submit
- Validasi ukuran file (maksimal 5MB)
- Format yang didukung: JPG, PNG

### 2. **Upload Galeri Foto**
- Upload multiple foto sekaligus (maksimal 5 foto)
- Preview semua foto yang dipilih
- Hapus foto individual dari galeri
- Validasi ukuran dan jumlah file

### 3. **UI/UX Improvements**
- Form upload yang intuitif dengan label yang jelas
- Preview foto real-time sebelum submit
- Progress indicator saat upload
- Error handling untuk validasi file

## 🔧 Technical Details

### Frontend Changes
- **File**: `frontend/src/components/dashboard/PemilikDashboard.vue`
- **Changes**:
  - Menambahkan input file untuk foto utama dan galeri
  - Implementasi FormData untuk multipart upload
  - Preview foto menggunakan URL.createObjectURL()
  - Validasi ukuran file (max 5MB)
  - Manajemen state untuk multiple file upload

### Backend Compatibility
- Endpoint sudah mendukung upload foto: `POST /api/pemilik/pesantren`
- Menggunakan `@fastify/multipart` untuk handling file upload
- Foto disimpan di direktori `uploads/pesantrenImages/`
- Data foto disimpan di database (foto_utama & foto_galeri JSON)

## 📋 Acceptance Criteria
- [ ] Pemilik bisa upload foto utama saat membuat pesantren baru
- [ ] Pemilik bisa upload galeri foto (max 5 foto)
- [ ] Preview foto berfungsi dengan benar
- [ ] Validasi ukuran file (max 5MB) bekerja
- [ ] Foto tampil di detail pesantren setelah upload
- [ ] Edit mode menampilkan foto yang sudah ada
- [ ] Hapus foto dari galeri berfungsi
