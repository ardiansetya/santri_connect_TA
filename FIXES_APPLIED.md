### 5. **Incomplete Data Loading in Dashboard** (MEDIUM)
**Status:** ✅ FIXED

**File Modified:**
- `frontend/src/components/dashboard/PendaftarDashboard.vue`

**Changes:**
```javascript
// Before: Just disabled loading without fetching data
onMounted(async () => {
  loading.value = false
})

// After: Properly fetches data and updates stats
onMounted(async () => {
  loading.value = true
  try {
    const { data } = await admin.getPendaftaran()
    pendaftaran.value = data.data || []
    stats.value = {
      pending: pendaftaran.value.filter(p => p.status === 'pending').length,
      diterima: pendaftaran.value.filter(p => p.status === 'diterima').length,
      total: pendaftaran.value.length
    }
  } catch (error) {
    console.error('Failed to load pendaftaran data:', error)
  } finally {
    loading.value = false
  }
})
```

**Impact:** Dashboard now displays actual data instead of empty state.
